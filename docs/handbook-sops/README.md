# Handbook SOP Library

This folder stores SOP drafts intended for inclusion in the operations handbook.

## Purpose
- Keep step-by-step operational procedures in one place.
- Reuse approved SOPs during handbook assembly and onboarding.
- Version SOP updates alongside code changes.

## File Naming Convention
- Use lowercase and hyphenated names.
- Prefix with process area when useful.
- Example: `group-reassignment-sop.md`
